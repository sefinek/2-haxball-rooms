# 2 HaxBall rooms ♿⚽
Launches two HaxBall rooms with the specified name and LAT/LON from the `.env` file. And that's all.

Uruchamia dwa pokoje HaxBall o konkretnej nazwie i LAT/LON z pliku `.env`. I to wszystko.
